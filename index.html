Arbetslivserfarenhet.

  • CGI - Förbättring av en träningsloggare Ett examensarbete utfördes hos IT-konsultföretaget CGI som använder en egenutvecklad applikation, kallad CGIMoving, som erbjuder motionskampanjer för sina anställda för att uppmuntra till en hälsosam livsstil. CGIMoving är ett webbsystem för loggning av fysiska träningsaktiviteter, där användare frivilligt kan delta och registrera sina aktiviteter och tjäna priser. En full-stack produktförbättring gjordes enligt en Scrum-metod, där en uppsättning av valda backlog features implementerades. Verktyg och tekniker som användes var ASP.NET-webbutvecklings framework och designmönstret Model-View-Controller i utvecklingsmiljön Visual Studio. Azure DevOps användes för versionshantering och för utveckling av webbapplikation användes ASP.NET, C#, HTML, CSS, JavaScript och MySQL databas. Jan 2020 - Juni 2020
  • Emiweb - Utveckling av sökmotor med användargränssnitt EmiWeb är en ideell förening med säte i Karlstad. Föreningen driver en webbtjänst för släktforskning, med fokus på utvandring från Sverige till USA under 1800 talet. Föreningen hade planerat att utveckla en ny version av webbtjänsten som innefattar en ny sökmotor byggd med modernare teknik - Elasticsearch som kärna. En full-stack utveckling utfördes i ett projektarbete. Webbsidan byggdes med HTML, CSS samt JavaScript som använder Ajax för dynamiskt innehåll. Backend utvecklades i Java med Spring Boot Framework. Den grundläggande databasen är MySQL med tabeller redan fyllda med kundens data. Okt 2019 - Jan 2020
